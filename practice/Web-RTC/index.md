## WebRTC 技术特点

WebRTC (Web Real-Time Communication) 示例展示了：

- **实时通信**：浏览器间点对点实时音视频通信
- **多线程处理**：使用 Web Workers 处理 WebRTC 数据流
- **媒体处理**：音视频流的获取、处理和传输
- **网络连接**：P2P 连接建立和数据通道管理

*WebRTC 是现代 Web 实时通信的核心技术，广泛应用于视频会议、在线教育、游戏等场景*
